📊 ALERT TYPES IMPLEMENTED (5)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. PRICE
     Purpose: Simple price threshold alerts
     Use Case: Buy/sell price targets
     Formula: current_price [>, <, >=, <=] target_value
     Example: AAPL > 150.00

  2. PERCENTAGE_CHANGE
     Purpose: Track relative movement
     Use Case: Momentum detection, adaptive to volatility
     Formula: |((current - last) / last) × 100| >= target
     Example: 5% move triggers alert
     Auto-Updates: last_price after each check

  3. VOLUME_SPIKE
     Purpose: Detect unusual trading activity
     Use Case: Institutional buying/selling, news reactions
     Formula: current_volume > (average_volume × target_value)
     Example: 1.5x normal volume triggers alert
     Data: Uses 20-day average volume

  4. CRASH
     Purpose: Downside protection
     Use Case: Stop-loss detection, price drop alerts
     Formula: percentage_drop <= target_value
     Example: 10% drop triggers alert
     Auto-Updates: last_price after each check

  5. CUSTOM
     Purpose: Extensible foundation
     Use Case: Future complex conditions
     Current: Works like PRICE alert
     Future: Can implement custom indicators, multi-factor rules
